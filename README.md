## 我的使用小技巧

用了这么久,我总结了几个让FlowSavvy更好用的小窍门:

**设置"缓冲时间"**:在任务之间留点休息时间,不要把日程排得太满。这样即使某个任务超时了,也不会影响后面的安排。

**善用"inbox"功能**:想到什么事,先扔到inbox里,不用马上设置细节。等有空了再回来补充截止日期和时长。

**区分"固定任务"和"自动任务"**:会议这种必须在特定时间的,设为固定任务;写报告这种灵活的,让FlowSavvy自动安排。

**定期review**:每周花5分钟看看下周的安排,及时调整不合理的地方。
